Description<br>
===========<br>

The program waits for the user to enter a person’s name.
It prints the person’s initials<br>

<strong>Compile the program</strong>

$ make initials

<strong>Usage</strong><br>

$ ./initials<br>
Pascal Blaise<br>
PB<br>

$ ./initials<br>
   roberto   Slinas beloko<br>
RSB<br>
